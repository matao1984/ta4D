# Tilt-averaged 4D-STEM acquisition and processing

`ta4D.py`: The class for acquisition. Needs TFS AutoScript
`ta4D_process.py`: The class for post-processing, e.g., align the scan positions and all diffraction patterns, and make averaged scan dataset

See `ta4D-process` notebook for the tutorial.
